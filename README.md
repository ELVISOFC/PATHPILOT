# PathPilot

PathPilot is an automated career trajectory & market analysis tool for salaried professionals.

## Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS

## Getting Started
1. `cd path-pilot`
2. `npm install`
3. `npm run dev`

## Value Proposition
PathPilot provides low-cost, automated career analysis, helping mid-to-senior level employees make informed decisions about their career path and regional moves at a fraction of the cost of traditional coaching.
